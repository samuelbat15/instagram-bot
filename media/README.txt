DOSSIER MÉDIAS — STAR BOXE COACHING
=====================================

Dépose ici tes photos et vidéos de la salle.

FORMATS ACCEPTÉS :
  Vidéos : .mp4, .mov
  Photos : .jpg, .jpeg, .png

CONVENTION DE NOMMAGE (recommandée) :
  Le bot cherche d'abord un fichier dont le nom contient le mot-clé du post.

  Exemples :
    boxing_sac_01.mp4       → utilisé pour les posts sur le sac
    boxing_sac_02.mp4
    coach_mitaines_01.mp4   → utilisé pour les posts coaching
    groupe_cours_01.jpg     → utilisé pour les posts cours collectifs
    transformation_01.jpg   → utilisé pour les posts témoignages
    nutrition_repas_01.jpg  → utilisé pour les posts nutrition
    ambiance_salle_01.mp4   → utilisé pour les posts ambiance

  Si aucun fichier ne correspond au mot-clé, un fichier aléatoire est choisi.

RÉSOLUTION RECOMMANDÉE :
  Format portrait 9:16 (ex: 1080x1920) pour un rendu optimal en Reel.
  Les autres formats sont automatiquement recadrés.
